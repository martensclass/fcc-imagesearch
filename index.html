FreeCodeCamp Image Search JSON Layer

How It Works

Add the path /api/search/<search term> to this url and get a json list of image results from IMGUR for that term. For example: 

           https://serene-temple-56606.herokuapp.com/api/search/coffee

..will return a json list of coffee images from IMGUR

Offestting Results

You can also offset the results and start at a different page by adding a page value BEFORE the search term as in: /api/search/page/<search term>

          https://serene-temple-56606.herokuapp.com/api/search/3/dogs

..will return a json list of dog images from IMGUR, starting at page 3 of the results. If you enter an invalid page value, the server will respond with an error.

To see the list of past searches add the path '/api/latest' to the url. Or what the heck, just click here to see a list of past searches.
